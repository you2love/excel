第五章：图表制作

5.1 图表基础

图表是数据可视化的重要工具，能够将复杂的数据以直观的方式呈现出来。

为什么要使用图表

  • 直观展示：将抽象的数据转换为视觉图形
  • 发现趋势
  • 突出重点：强调关键信息和异常值
  • 便于理解：让非专业人士也能理解复杂数据
  • 专业展示：提升报告和演示的专业性

图表的基本组成

  • 图表区：整个图表的背景区域
  • 绘图区：显示数据的实际区域
  • 数据系列：图表中的数据集合
  • 坐标轴：X轴（分类轴）和Y轴（数值轴）
  • 图例：说明数据系列含义的说明
  • 标题：图表的主标题和轴标题
  • 数据标签：显示具体数值的标签
  • 网格线：帮助读取数值的参考线

5.2 创建图表

Excel提供了多种创建图表的方法。

使用功能区创建图表

  1. 选中要制图的数据区域（包含标题）
  2. 点击"插入"选项卡
  3. 在"图表"组中选择合适的图表类型
  4. 选择具体的图表样式

使用快速分析创建图表

  1. 选中数据区域
  2. 点击选区右下角出现的"快速分析"按钮
  3. 点击"图表"选项卡
  4. 选择合适的图表类型

快捷键创建图表

  • Alt+F1：在当前位置创建默认图表
  • F11：在新工作表中创建默认图表
小技巧：选中数据后按F11可以快速在单独的工作表中创建图表，适合复杂图表的制作。

5.3 常用图表类型

Excel提供了丰富的图表类型，选择合适的图表类型是制作优秀图表的关键。

柱状图

适用场景：比较不同类别的数值大小

  • 簇状柱形图：并排显示多个数据系列
  • 堆积柱形图：显示各部分占总体的比例
  • 百分比堆积柱形图：显示各部分占总体的百分比

折线图

适用场景：显示数据随时间变化的趋势

  • 折线图：基本的趋势线
  • 带数据标记的折线图：显示具体数据点
  • 堆积折线图：显示累计趋势

饼图

适用场景：显示各部分占总体的比例

  • 饼图：基本的比例显示
  • 环形图：中心的空洞可以显示总计
  • 复合饼图：突出显示特定部分
注意：饼图不宜显示过多的类别，建议不超过5-7个，否则难以阅读。

条形图

适用场景：与柱状图类似，但类别名称较长时更合适

面积图

适用场景：显示数量随时间的变化，同时显示总量的变化趋势

散点图

适用场景：显示两个变量之间的关系和相关性

  • 散点图：基本的XY关系图
  • 带平滑线的散点图：显示趋势线

直方图

适用场景：显示数据的分布情况

箱线图

适用场景：统计分析，显示数据的中位数、四分位数和异常值

瀑布图

适用场景：显示初始值如何受正负值影响达到最终值

漏斗图

适用场景：显示数据在多个阶段递减的情况，如销售漏斗

组合图

适用场景：结合不同类型的图表显示多个数据系列

  • 簇状柱形图-折线图：结合比较和趋势
  • 堆积面积图-簇状柱形图：显示构成和比较

5.4 图表设计

创建图表后，可以通过设计选项卡进行美化和定制。

更改图表类型

  1. 点击图表
  2. 点击"图表设计"选项卡
  3. 点击"更改图表类型"
  4. 选择新的图表类型

应用图表样式

  1. 点击图表
  2. 点击"图表设计" > "图表样式"
  3. 选择合适的样式

切换行/列

点击"图表设计" > "切换行/列"，交换X轴和Y轴的数据。

选择数据

点击"图表设计" > "选择数据"，可以：

  • 添加或删除数据系列
  • 编辑数据系列的来源
  • 编辑水平分类轴标签
  • 切换行列数据
高级技巧：可以直接拖动图表中的数据区域手柄来调整图表显示的数据范围。

5.5 图表元素

Excel图表包含多个可自定义的元素。

添加图表元素

  1. 点击图表
  2. 点击图表右上角的"+"按钮
  3. 勾选要添加的元素

常用图表元素

  • 坐标轴标题：说明坐标轴含义
  • 图表标题：图表的主标题
  • 数据标签：显示数据点的具体值
  • 数据表：在图表下方显示数据表
  • 误差线：显示数据的误差范围
  • 网格线：辅助读取数值
  • 图例：说明数据系列
  • 趋势线：显示数据趋势
  • 涨/跌柱线：在折线图中显示变化

格式化图表元素

  1. 点击要格式化的图表元素
  2. 右键点击，选择"格式化..."
  3. 在格式化面板中进行设置

5.6 趋势线和误差线

趋势线和误差线是数据分析的重要工具。

趋势线类型

  • 线性趋势线：显示线性关系
  • 指数趋势线：显示指数增长
  • 对数趋势线：显示对数关系
  • 多项式趋势线：显示复杂曲线
  • 移动平均趋势线：平滑数据波动

添加趋势线

  1. 点击数据系列
  2. 点击图表元素"+"按钮
  3. 勾选"趋势线"
  4. 选择趋势线类型

误差线

误差线显示数据的不确定性范围：

  • 固定值误差线
  • 百分比误差线
  • 标准偏差误差线
  • 标准误差误差线

5.3 迷你图

迷你图是在单元格中创建的小型图表，适合显示趋势概览。

创建迷你图

  1. 选中要放置迷你图的单元格
  2. 点击"插入"选项卡
  3. 选择迷你图类型：折线图、柱形图、盈亏图
  4. 选择数据范围
  5. 点击确定

迷你图类型

  • 折线图：显示趋势变化
  • 柱形图：显示数值比较
  • 盈亏图：显示正负值

迷你图设置

在"迷你图设计"选项卡中可以：

  • 编辑数据范围
  • 更改图表类型
  • 显示标记点
  • 设置高低点颜色
  • 设置坐标轴
小技巧：将迷你图与数据表格配合使用，可以快速创建仪表板风格的报告。

5.8 动态图表

动态图表可以根据数据变化自动更新，适合实时数据分析。

基于表格的动态图表

  1. 将数据区域转换为表格（Ctrl+T）
  2. 创建基于表格的图表
  3. 在表格中添加新数据，图表自动更新

基于命名范围的动态图表

  1. 定义动态命名范围（使用OFFSET或INDEX函数）
  2. 创建基于命名范围的图表
  3. 数据变化时图表自动更新

5.9 图表美化技巧

专业的图表不仅要有准确的数据，还要有良好的视觉效果。

颜色搭配

  • 使用协调的颜色方案
  • 避免使用过多颜色（建议不超过5-6种）
  • 使用对比色突出重要数据
  • 保持与品牌颜色一致

字体选择

  • 使用清晰易读的字体
  • 标题字体略大于标签字体
  • 保持字体风格一致

布局设计

  • 保持图表简洁，避免过度装饰
  • 合理留白，避免拥挤
  • 确保所有元素清晰可读
  • 删除不必要的网格线和图例
设计原则：好的图表应该简单、清晰、准确，让读者能够在3秒内理解图表传达的信息。

5.10 图表模板

可以将设计好的图表保存为模板，方便重复使用。

保存图表模板

  1. 点击要保存为模板的图表
  2. 右键点击，选择"另存为模板"
  3. 输入模板名称
  4. 点击保存

应用图表模板

  1. 创建新图表
  2. 点击"更改图表类型"
  3. 点击"所有图表"选项卡
  4. 点击"模板"
  5. 选择要应用的模板

5.13 图表常见问题

如何处理过多的数据系列？

建议：

  • 使用次坐标轴区分不同量级的数据
  • 拆分为多个图表
  • 使用小倍数图表（small multiples）

如何选择合适的图表类型？

选择原则：

  • 比较：柱状图、条形图
  • 趋势：折线图
  • 占比：饼图、环形图
  • 关系：散点图
  • 分布：直方图、箱线图

如何让图表更专业？

专业图表要素：

  • 清晰的标题和说明
  • 适当的数据标签
  • 简洁的网格线
  • 协调的颜色方案
  • 合理的图例位置
