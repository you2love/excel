# Excel 学习中心

一个现代化的Excel学习资料网站，采用纯静态HTML/CSS/JavaScript技术栈，提供系统化的Excel学习教程。

## 项目概述

本项目是一个完整的Excel学习网站，涵盖Excel的功能和核心操作，适合初学者到中级用户系统学习Excel。

## 技术栈

- **HTML5**：语义化标签，结构清晰
- **CSS3**：现代样式设计，响应式布局
- **JavaScript**：交互功能，用户体验优化

## 项目结构

```
excel-tutorial/
├── index.html              # 网站首页
├── css/
│   └── style.css           # 主样式文件
├── js/
│   └── main.js             # 主JavaScript文件
├── pages/                  # 教程页面
│   ├── introduction.html    # 第一章：Excel界面介绍
│   ├── basics.html         # 第二章：Excel基础操作
│   ├── formatting.html     # 第三章：格式化与美化
│   ├── formulas.html       # 第四章：公式与函数
│   ├── charts.html         # 第五章：图表制作
│   └── pivottables.html    # 第六章：数据透视表
├── images/                 # 图片资源目录
└── README.md              # 项目说明文件
```
